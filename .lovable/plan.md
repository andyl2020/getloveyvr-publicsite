

# GetLoveYVR — Singles Events Website

## Overview
A simple, SEO-optimized single-page website for GetLoveYVR — a Vancouver-based singles event brand that hosts fun, activity-based dating events. The site targets newcomers who don't know the brand yet.

## Design Direction
- **Palette**: Olive green (from the logo) + warm cream/off-white background + dark accents. Clean and modern.
- **Typography**: Modern, friendly sans-serif (Space Grotesk headings + DM Sans body)
- **Vibe**: Fun but professional. Minimal, not cluttered. Lots of whitespace.
- **Logo**: The uploaded heart/bridge/mountains logo placed in the header

## Page Sections (single page, scrollable)

### 1. Hero
- Logo + "GetLoveYVR" brand name
- Tagline: "Making modern dating fun again 💖"
- Subtitle: "8 events. 10 weeks. Will you find love?"
- CTA button → scrolls to events / links to Flock Social signup
- Social links (TikTok, Instagram)

### 2. About
- Brief description: fun, low-pressure singles events in Vancouver — meet real people, not just swipe on them
- Activity-based events (boxing, improv, baking, painting, etc.)
- How it works (show up → have fun → connect)

### 3. Upcoming Events — Calendar View
- A clean monthly calendar (April–June 2026) showing event dates
- Events highlighted on their dates with event names
- Clicking/hovering an event shows details
- Events:
  - Apr 26: Boxing
  - May 3: Improv
  - May 10: Baking
  - May 24: Painting
  - May 31: Social / Bingo (tentative)
  - Jun 7: TBD (tentative)

### 4. Join / CTA
- "Ready to find love?" section
- Link to Flock Social community page for signups
- Social media links (TikTok, Instagram)

### 5. Footer
- Brand name, social links, Vancouver CA

## SEO
- Semantic HTML with proper heading hierarchy
- Meta title: "GetLoveYVR | Fun Singles Events in Vancouver"
- Meta description targeting "singles events Vancouver" keywords
- Open Graph tags

- Fully static, no backend needed
- Logo copied into project assets
- Responsive (mobile-first)
